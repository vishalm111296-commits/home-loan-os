# Sessions Folder

Stores a log of every work session.

## File Naming
```
sessions/
├── README.md
├── 2026-06-21-session.md
├── 2026-06-22-session.md
└── 2026-06-23-session.md
```

## Session Log Format
```
Date: YYYY-MM-DD
Account used: A / B / C / D
Model: Claude / ChatGPT / Gemini / Perplexity
Goal: [what we tried to do]
Completed: [what actually got done]
Result: [one line summary]
Files updated: [list]
Next session: [first thing to do next time]
```

## Rule
AI generates this at end of session automatically.
You paste it here as a new dated file.
Never overwrite old sessions.
